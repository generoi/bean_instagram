Instagram Bean
==============

Provide instagram feed bean type using the Instafeed.js library.

Installation
------------
1. Enable
2. Download the latest version of instafeed
   (https://github.com/stevenschobert/instafeed.js) and place it it
   libraries/instafeed so that the file libraries/instafeed/instafeed.min.js
   exists.
3. Visit block/add/instagram and add your first bean instance.
